# Lesson 1
## Conversation
|    | Deeni | English |
|---|---|---|
| Erin  | Cheela!  | Hello!   |
|   | Eelen waa ‘vm-nvsh-shi. | My name is Erin.   |
| Judy  | Cheela!  |  Hello! |
|   | Ch’uudi waa ‘vm-nvsh-shi.  |  My name is Judy. |

## Grammar Notes
- _Waa_ means _in that way. _
- _‘Vm-nvsh-shi_ means _I am called._
- [Name] waa ‘vm-nvsh-shi :arrow_right: [Name] is the way I am called.
